# Prompt de Création de Logo — PassScan

## Contexte de l'application

**PassScan** est une application multiplateforme (iOS + Web) de scan de documents d'identité (zone MRZ des passeports et cartes d'identité) destinée à la gestion d'entrées événementielles (listes VIP, blacklists, contrôle d'accès). L'application est liée à la marque "Village du Soir" (événementiel nocturne à Genève, Suisse). Le créateur est Safwan Abdirahman, entrepreneur suisse spécialisé en technologie et innovation.

---

## Prompt Principal (Génération IA — Midjourney / DALL-E / Ideogram)

```
Design a modern, professional logo for "PassScan", a mobile and web application for scanning identity documents (passport MRZ zones) and managing event access control (VIP lists, blacklists, guest management).

The logo must combine these symbolic elements:
- A stylized passport or ID document silhouette (subtle, not literal)
- A scanning beam or scan line effect (representing the MRZ scanning technology)
- A shield or checkmark motif (representing security, verification, and trust)
- Optional: a subtle "P" and "S" monogram integration

Style direction:
- Clean, minimal, geometric design language
- Tech-forward and premium feel — think fintech/security app aesthetics
- Must work as both a full wordmark AND a standalone icon/symbol
- Rounded corners friendly (iOS app icon compatible)
- No gradients in the core mark (gradients only in optional variations)

Color palette:
- Primary: Deep electric blue (#2563EB) — trust, technology, security
- Secondary: Dark charcoal (#1F2937) — professionalism, elegance
- Accent: Bright cyan (#06B6D4) for the scan beam element
- The logo must also work in pure white (on dark backgrounds) and pure black (on light backgrounds)

Typography (for wordmark version):
- "Pass" in medium weight, "Scan" in bold weight
- Modern sans-serif typeface (similar to Inter, SF Pro, or Satoshi)
- Clean letter spacing, tech-oriented feel
- The word "Scan" should feel slightly more dynamic/emphasized

Technical constraints:
- Scalable from 16x16 favicon to billboard size
- Recognizable at 29x29px (smallest iOS icon)
- Must remain legible and distinctive at all sizes
- No fine details that disappear at small scales
- Square-safe composition (for app icons)

Mood: Professional, trustworthy, innovative, Swiss precision, nightlife-event-ready
Avoid: Clip art style, overly complex details, literal passport imagery, childish elements, generic shield icons
```

## Déclinaisons Demandées

### 1. Icône App (Symbol Mark)
```
Create a square app icon version of the PassScan logo. Standalone symbol only, no text.
Must be perfectly readable at 1024x1024px and still recognizable at 16x16px.
Rounded corners (iOS superellipse mask compatible).
Background: solid deep blue (#2563EB) with white/cyan symbol.
Alternative: dark charcoal (#1F2937) background with blue/cyan symbol.
The icon should feel premium and immediately suggest "document scanning + security".
Clean, geometric, minimal — 2-3 visual elements maximum.
```

### 2. Wordmark Complet (Logo Horizontal)
```
Create a horizontal wordmark logo for "PassScan".
Symbol/icon on the left, "PassScan" text on the right.
Balanced proportions — the icon should be roughly the height of the text.
Versions needed:
- Full color on white background
- Full color on dark background (#111827)
- Monochrome white (for dark overlays/photos)
- Monochrome black (for light backgrounds and print)
Aspect ratio approximately 3:1 to 4:1 (wide format).
```

### 3. Logo Empilé (Vertical / Stacked)
```
Create a stacked/vertical version of the PassScan logo.
Symbol centered on top, "PassScan" text centered below.
Aspect ratio approximately 1:1.2 (slightly taller than wide).
Ideal for: splash screens, loading screens, centered layouts, social media avatars.
Same color variations as the horizontal version.
```

### 4. Favicon & Micro-Icône
```
Create an ultra-simplified version of the PassScan symbol for use as a favicon.
Must be perfectly clear at 16x16px, 32x32px, and 48x48px.
Maximum 1-2 visual elements. Bold strokes, high contrast.
Consider: just the scan beam + initial "P" letter, or the shield + scan line.
No fine details whatsoever.
Transparent background version + solid background version.
```

### 5. Monogramme / Initiales
```
Create a "PS" monogram for PassScan.
Letters "P" and "S" intertwined or cleverly combined in a geometric style.
Should subtly evoke scanning or security.
Use as: watermark, embossed element, premium branding touch.
Works in single color. Clean geometric construction.
```

### 6. Version "Village du Soir" (Co-branding)
```
Create a co-branded version: "PassScan" logo with "by Village du Soir" or
"powered by PassScan" subtitle.
The subtitle should be significantly smaller (about 30-40% of main text size).
Subtle, elegant, secondary hierarchy.
This version is for the event-facing side of the product.
Add a subtle nightlife/evening touch — maybe a crescent moon accent or
star element near "Village du Soir", while keeping the tech feel of PassScan.
```

### 7. Version Dark Mode / Light Mode
```
Produce two optimized versions:
- LIGHT MODE: Dark logo (#1F2937 primary) with blue accent, on white/light gray
- DARK MODE: White/light logo with cyan (#06B6D4) scan accent, on dark (#0F172A)
Both must feel equally premium and intentional (not just an inversion).
The dark mode version can use slightly more glow/luminosity on the scan element.
```

### 8. Version Splash Screen (iOS & Web)
```
Create a centered, dramatic splash screen composition:
- PassScan logo (stacked version) centered
- Subtle radial gradient background: from deep blue center to darker edges
- Optional: very subtle scan-line animation suggestion (horizontal light streak)
- Premium, immersive, app-launch feel
- Resolution: 1290x2796px (iPhone 15 Pro Max) and 1920x1080px (web)
```

### 9. Social Media Kit
```
Create PassScan logo variations optimized for:
- Profile picture (circular crop safe): 400x400px — use the symbol mark
- Banner/cover: 1500x500px (Twitter/X), 1584x396px (LinkedIn) — horizontal wordmark centered
- Story format: 1080x1920px — stacked logo on branded background
Ensure the logo has enough padding to survive circular cropping on all platforms.
```

### 10. Print & Merch
```
Create a version suitable for print and merchandise:
- High contrast, works in 1-2 spot colors (Pantone-ready)
- Vector-clean edges (no raster effects, no gradients)
- Embossing/debossing friendly version (single depth)
- Works on: business cards, t-shirts, lanyards, event badges, stickers
- Minimum size: must remain legible at 15mm width
```

## Notes pour le Designer / l'IA Générative

1. **Cohérence avant tout** : toutes les déclinaisons doivent être immédiatement identifiables comme appartenant à la même famille visuelle.
2. **Le scan beam est l'élément signature** : c'est lui qui différencie PassScan d'un logo de sécurité générique. Il doit apparaître dans chaque version (même de manière subtile).
3. **Swiss quality** : le logo doit refléter la précision et l'élégance suisse — propre, mesuré, sans excès.
4. **Double lecture** : le logo doit fonctionner aussi bien dans un contexte tech/startup que dans un contexte événementiel/nightlife.
5. **Test à toutes les tailles** : chaque version doit être testée de 16px à plein écran avant validation.
